# Workspace — Three-Layer Agent System

> This file is maintained by the agent system. You can edit it freely.
> Agents read this file to understand what has been done in this workspace.

## Structure

| Path | Purpose |
|------|---------|
| `workspace/current/files/` | Agent-created and task-produced files |
| `workspace/current/memory/` | Long-term memory (do not edit manually) |
| `workspace/current/shared/` | Cross-agent peer cache |
| `workspace/logs/agent.log` | All agent activity (append across sessions) |
| `workspace/logs/structured.ndjson` | Structured logs (machine-readable) |
| `workspace/sessions/` | Per-session summaries |
